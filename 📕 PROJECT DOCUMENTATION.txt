📕 PROJECT DOCUMENTATION
Tiny Path – Private Micro Social Network
1️⃣ Project Overview

Project Name: Tiny Path
Purpose:
Recreate the aesthetic and lightweight functionality of the original Path social app for a small private friend group.

Goal:
Create a mobile-first social feed that supports:

Text posts

Photo posts

Location check-ins

Realtime updates

Persistent usernames (browser memory)

Minimal friction (no accounts or login)

Deployment Platform: Netlify
Database: Firebase Firestore
Image Hosting: Cloudinary
Reverse Geocoding: OpenStreetMap Nominatim API

2️⃣ Architecture Overview

Frontend (Static Site):

HTML

CSS

Vanilla JavaScript (ES Modules)

Backend Services:

Firebase Firestore (Realtime database)

Cloudinary (Image storage)

OpenStreetMap API (Reverse geocoding)

Hosting:

Netlify (drag-and-drop deploy)

No backend server required.

3️⃣ Features Implemented
📝 Text Posts

Users can:

Enter a username

Write text

Post to feed

Stored in Firestore with:

{
  text: string,
  username: string,
  imageUrl: string | null,
  location: string | null,
  createdAt: timestamp
}
📷 Photo Uploads

Flow:

User taps 📷

File input opens

Image uploads to Cloudinary

Cloudinary returns secure_url

URL saved to Firestore

Image rendered in feed

Why Cloudinary?

Avoids Firebase paid storage tier

Free hosting

Simple unsigned preset

CDN optimized

📍 Location Check-Ins (Polished Version)

Flow:

User taps 📍

Browser asks for geolocation permission

Coordinates retrieved

OpenStreetMap Nominatim reverse geocoding API called

Extract:

city

state

Save as formatted string:

"Austin, Texas"

Render as gray location pill in feed

Location UI:

Gray rounded pill above text

Subtle, nostalgic Path style

🔄 Realtime Feed

Uses Firestore:

onSnapshot(query(...))

This enables:

Instant updates

Multi-user live feed

No page refresh required

🧠 Username Memory

Stored using:

localStorage.setItem("tinyPathUsername", username)

When page reloads:

Username auto-populates

No login required

Low-friction identity

🖼 Fullscreen Image Modal

When image clicked:

Modal overlay appears

Image enlarged

Tap outside or ❌ closes

Bug fix included:

Modal default display set to none

.hidden utility class enforced

1️⃣2️⃣ Project Summary

Tiny Path is a lightweight, mobile-first, serverless private social network inspired by the original Path app. It supports text posts, image uploads, and location check-ins with realtime updates, using Firebase Firestore for storage and Cloudinary for media hosting.

Deployed via Netlify as a static site, the app requires no backend server and is optimized for simplicity and nostalgia.
